# Darts Perfect - 501 Steel Tip Scorer

## 배포 방법 (처음이라도 할 수 있어요!)

### 1단계: GitHub에 코드 올리기

1. GitHub.com 로그인
2. 오른쪽 위 **+** 버튼 → **New repository** 클릭
3. Repository name: `darts-perfect` 입력
4. **Public** 선택
5. **Create repository** 클릭
6. "uploading an existing file" 링크 클릭
7. 이 폴더의 모든 파일을 드래그해서 올리기
8. **Commit changes** 클릭

### 2단계: Vercel에서 배포하기

1. vercel.com 접속
2. **Sign up with GitHub** 로 가입/로그인
3. **Add New** → **Project** 클릭
4. `darts-perfect` 저장소 선택 → **Import**
5. Framework: **Create React App** 자동 감지됨
6. **Deploy** 클릭
7. 2-3분 후 URL이 나옵니다! (예: darts-perfect.vercel.app)

### 완료!
그 URL을 태블릿 브라우저에서 열면 바로 사용 가능합니다.
홈 화면에 추가하면 앱처럼 사용할 수 있어요.
